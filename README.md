# ⚙️ University Engineering Archive

Welcome to my digital engineering vault. I am a Mechanical Engineering student passionate about bridging the gap between theoretical math, physical manufacturing, and professional technical communication. 

This repository serves as an open-source archive of my university coursework, independent mechatronics projects, and computational research.

## 🗂️ Repository Structure

* **`/Semester_Notes`** — Comprehensive, mathematically rigorous lecture notes and lab reports typeset in LaTeX.
* **`/CAD_Models`** — Parametric 3D designs, assemblies, and files prepped for rapid prototyping (Fusion 360 / SolidWorks).
* **`/Python_Analysis`** — Scripts for kinematic simulations, data logging, and experimental data optimization.
* **`/Mechatronics`** — Arduino (C++) code for sensor integration, PID control loops, and hardware testing.

## 🛠️ Core Toolstack
* **Engineering & Simulation:** Python, EES, MATLAB
* **Design & Manufacturing:** Fusion 360, 3D Printing (FDM)
* **Technical Publishing:** LaTeX (Overleaf), Adobe Creative Cloud (Illustrator, InDesign)
